# Build

1. Clone repo

   `git clone https://github.com/StarPandaBeg/DayNNight.git`

1. Create a directory for cmake , e.g.

   `mkdir build`

1. Run cmake to create config cache

   `cmake -B build -DCMAKE_BUILD_TYPE=Release`

1. Run cmake build

   `cmake --build build --config Release`
